About Easy Math Learning

Easy Math Learning is a project destinated to help young students to improve their skills in solving math problems. Our tools are designed to help you to master from basic to complex math operations by using three keys indicators:

  • Average of Hits: Naturally, a person knows how to solve math problems if can find the right answer. Average of Hits value is given by the number of right answers divided by the number of tries.

  • Structure: When solving problems, reach the right answer is not enough, you must also know “how write” your answers correctly. You can see a set of templates explaining how to solve problems step by step in the section Lessons (coming soon).

  • Speed: while present a test the time becomes a very dangerous enemy. Be able to solve problems within the limit time could determine if you pass or not the test, even if you know how to solve the problem, that’s why to improve your speed every exercise keep a register of the time expended in each problem so you can work to improve it.

How This Work?

You can improve your time by using our free online calculator tool and timer. Unlike other calculators, this will not return the answer immediately but will wait until the user provides an answer, ones the answer is submitted the timer stops, and the answer is shown.

Try Out!

  1. Write the exercise using the calculator.
  2. When you are ready press the Start button.
  3. Write your answer and press Stop button.
  4. Compare the answers and improve your time.
